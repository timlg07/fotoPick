# fotoPick
View your RAW and JPEG photos combined, delete bad ones and pick your favorites. 
Perfect for a quick preselect before loading them in Lightroom or similar.

## Install and Run
```
npm install
npm run-script demo
```

## Build
```
npm run-script dist
```
